# AI-Powered ATS Resume Analyzer 🚀

An AI-powered ATS Resume Analyzer that evaluates resumes against job descriptions — providing match percentage, missing keywords, skill improvement suggestions, and profile summary using LLaMA 3.3 70B via Groq API, built with Python and Streamlit.

## 🔍 Features

- **Tell me about the Resume** — HR-level evaluation of resume vs job description
- **How can I Improve my Skills** — Personalized skill improvement suggestions
- **Percentage Match** — ATS match score, missing keywords and profile summary

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API (LLaMA 3.3 70B)
- PyPDF2
- python-dotenv

## ⚙️ How to Run Locally

1. Clone the repo
```bash
git clone https://github.com/deepakarjunk-dak/AI-Powered-ATS-Resume-Analyzer.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Add your Groq API key in `.env`
```
GROQ_API_KEY=your_groq_api_key_here
```

4. Run the app
```bash
streamlit run app.py
```
